## GoPlay
GO - Fundamentals

1. Install Go
2. Install Sublime text
3. Go to GoPlay project folder
4. Set gopath              ==>    export GOPATH=`pwd`
5. Compile/Install Program ==>    go install hello
6. Run program             ==>    bin/hello
